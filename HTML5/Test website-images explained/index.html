Images In HTML:

Images used in a site should always be relevant. Never pick an image which is not relevant and acts as a waste on your website.

An image used on your website should always match the color scheme of your website. Nothing looks more horrible than mixed colors.

If you cant decide where to find relevant images and google is not accurate enough for than you can use different other webise.
fair warning: things on some websites will not be free. Best if you steal images from code or google it.

Images are added on website by the <img> tag.
In this <img> tag there are some attributes that are much necessary. These are:

  1. <src=" ">
    • This is used as the location finder for your image-better known as source. Whether they are online someplace or offline on your computer.
    • Online images are added by using the full address of the image. Example:
      <img src="https://www.seeklogo.net/wp-content/uploads/2013/11/facebook-flat-vector-logo-400x400.png">
    • Offline images are added the same way links are added to seperate pages in website. Example:
      <img src="images/fb_logo.png">
  2. <alt=" ">
    • This attribute is used to add an alternative text if the image fails to load or is missing from the location described.
  3. <width=" ">
    • This is used if you want to size the width of your image but don't want to use css or photoshop.
  4. <height=" ">
    • This is used if you want to size the height of your image but don't want to use css or photoshop.
